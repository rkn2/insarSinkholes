# ML Experiments Staging

This folder contains ML-layer additions moved out of the main app path to keep the base dashboard workflow clean.

- `dashboard/`: ML-conditioned dashboard/export variants preserved from this session.
- `scripts/`: LOEO/control/export scripts for ML validation experiments.
- `config/`: ML experiment split/control config files.

Main app files in repository root were restored to pre-ML-layer versions.
